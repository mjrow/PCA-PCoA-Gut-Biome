# PCA/PCoA Application for Analysis of Gut Microbiomes in Cancer Patients

## 📝 Project Statement
This project aims to demonstrate our understanding of PCA and PCoA, its importance in dimensionality reduction within datasets, and its application to real external datasets. We worked with gut microbiome data collected from cancer patients to demonstrate the usefullness of PCA/PCoA when comparing the gut microbiomes of cancer patients and healthy individuals.
